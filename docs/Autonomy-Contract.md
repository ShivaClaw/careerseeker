# CareerSeeker Autonomy Contract Placeholder

Status: draft product/legal placeholder. L1 only for v0.1.

CareerSeeker acts only within the autonomy level and rails the user chooses. Every material action is logged and reviewable.

## L1 Drafts Mode

CareerSeeker may discover jobs, score them, research companies, tailor application materials, run the Fabrication Gate, and create Gmail drafts for user review.

CareerSeeker may not send email, submit applications, answer novel recruiter questions, or book calendar events in L1.

## Non-Negotiable Rules

- The Fabrication Gate cannot be bypassed. Unsupported claims block before an application reaches READY.
- Novel free text in the user's voice is not auto-sent.
- Low-legitimacy or severe-red-flag postings may be shown but not acted on.
- User resume/profile data and OAuth tokens remain local to the Windows engine.
- Every state transition is written to the tamper-evident audit log.

## User Rails

Before any future L2/L3 mode, the user must explicitly configure:

- Maximum applications per day.
- Minimum score threshold.
- Current-employer and domain blocklist.
- Approved Answer Bank.
- Quiet hours.
- Calendar windows, if scheduling is enabled.
